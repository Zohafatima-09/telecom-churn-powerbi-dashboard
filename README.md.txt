# Telecom Customer Churn Analysis (Power BI)

# Project Overview
This dashboard analyzes customer churn patterns in the telecom industry using Power BI.

# Key Insights
- Total Customers: 4804
- Active Customers: 3148
- Churn Rate: 34%
- Highest churn in Month-to-Month contracts
- Electronic check users have higher churn
